How does it work?

It's really simple! We use the following formulas during a transaction:

$$\text{balance}_{\text{sender}} = \text{balance}_{\text{sender}} - \text{amount}$$ $$\text{balance}_{\text{receiver}} = \text{balance}_{\text{receiver}} + \text{amount}$$

And the source code is also very simple! 


def perform_transaction(sender, receiver, amount):
    accounts[sender]['balance'] -= amount
    accounts[receiver]['balance'] += amount
